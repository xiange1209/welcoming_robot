==========================================================
 智慧銀行迎賓機器人 常用指令（樹莓派自行編譯版）
 適用：RPi4 4GB/8GB + Ubuntu 24.04 + ROS 2 Jazzy
 佈局：~/smartnav_ws = repo 根（廠商套件 src/、自有套件 smartnav-bot/src/）
 另一版本見「常用指令_虛擬機編譯版.txt」（RPi4 免編譯）
==========================================================

0. 每次開新終端（建議寫進 ~/.bashrc）
----------------------------------------------------------
source /opt/ros/jazzy/setup.bash
source ~/smartnav_ws/install/setup.bash

1. 取得程式碼（第一次）
----------------------------------------------------------
git clone https://github.com/xiange1209/welcoming_robot.git ~/smartnav_ws
cd ~/smartnav_ws
# 自動探索建圖需另 clone（一次性）：
git clone https://github.com/mertgulerx/frontier_exploration_ros2 smartnav-bot/src/frontier_exploration_ros2
# 之後更新：
git pull

2. 全量編譯（第一次，約 1~2 小時；RPi4 記憶體有限務必限制平行度）
----------------------------------------------------------
# 一次性：加 4G swap 防 OOM
sudo fallocate -l 4G /swapfile && sudo chmod 600 /swapfile && sudo mkswap /swapfile && sudo swapon /swapfile
# 一次性：裝依賴
sudo rosdep init; rosdep update
rosdep install --from-paths src smartnav-bot/src --ignore-src -r -y
sudo apt install ros-jazzy-navigation2 ros-jazzy-slam-toolbox ros-jazzy-cartographer-ros \
  ros-jazzy-robot-localization ros-jazzy-topic-tools ros-jazzy-joint-state-publisher \
  ros-jazzy-web-video-server ros-jazzy-rosbridge-server ros-jazzy-teleop-twist-keyboard \
  libuvc-dev libgoogle-glog-dev
# 編譯（限平行度）：
cd ~/smartnav_ws
MAKEFLAGS="-j2" colcon build --symlink-install --executor sequential --parallel-workers 1

3. 日常增量編譯（只編你改過的套件，幾秒~幾分鐘）
----------------------------------------------------------
colcon build --symlink-install --packages-select smartnav_llm     # 例：只改了 LLM
colcon build --symlink-install --packages-select smartnav_bringup

4. 底盤（senior_akm 高配阿克曼；一次性先跑 udev）
----------------------------------------------------------
# 一次性 udev（跑完重插 USB，應出現 /dev/wheeltec_controller）：
sudo bash src/turn_on_wheeltec_robot/wheeltec_udev.sh
# 單獨開底盤：
ros2 launch turn_on_wheeltec_robot turn_on_wheeltec_robot.launch.py
# 鍵盤遙控（車架高再測！）：
ros2 run teleop_twist_keyboard teleop_twist_keyboard

5. 雷達 / 相機（單獨測）
----------------------------------------------------------
ros2 launch turn_on_wheeltec_robot wheeltec_lidar.launch.py    # N10 光達 → /scan
ros2 launch turn_on_wheeltec_robot wheeltec_camera.launch.py   # astra 深度相機

6. 建圖 → 存圖 → 導航（wheeltec 官方鏈，自帶底盤，勿與 enable_chassis 同開）
----------------------------------------------------------
ros2 launch wheeltec_slam_toolbox online_async_launch.py       # 建圖（開車繞場）
ros2 launch wheeltec_nav2 save_map.launch.py                   # 存圖
ros2 launch wheeltec_nav2 wheeltec_nav2.launch.py              # 導航（自動載 senior_akm 參數）
# 注意：阿克曼不能原地旋轉，senior_akm 迴轉需約 1.6m 淨空

7. 智慧功能統一啟動（smartnav bringup）
----------------------------------------------------------
# 完整（視覺+語音+LLM+底盤+平板HMI+瀏覽器影像）：
ros2 launch smartnav_bringup smartnav.launch.py \
  ollama_base_url:=http://<筆電IP>:11434 model_name:=qwen2.5:3b enable_gpu:=false \
  enable_chassis:=true enable_navigation:=false enable_hmi:=true enable_web_video:=true
# 平板瀏覽器開：http://<RPi4-IP>:8081（HMI）；http://<RPi4-IP>:8080（影像列表）
# 語音改用麥克風陣列（離線命令詞）：加 audio_stack:=wheeltec
# LLM 改用輕量單輪對話：加 llm_stack:=wheeltec

8. 人臉註冊 / 除錯觀察
----------------------------------------------------------
ros2 service call /face_registration/register smartnav_msgs/srv/RegisterFace \
  "{person_name: '陳佳憲', gender: 'M', person_type: 'VIP', num_samples: 10}"
ros2 topic echo /face_recognition/result
ros2 topic echo /user_text
ros2 topic echo /llm_response
ros2 topic echo /odom          # 底盤里程計
ros2 topic echo /scan --once   # 雷達有無資料

9. 疑難排解速記
----------------------------------------------------------
- 開機 10 秒內底盤不理指令＝正常（韌體保護）
- 藍牙 APP 連著時 ROS 指令無效（韌體優先序），先斷 APP
- /scan 沒資料：核對雷達貼紙，wheeltec_param.yaml 的 lidar_type
  在 ls_N10_uart / ls_N10Plus_uart 之間切換
- 編譯 OOM 死機：確認 swap 有開、用第 2 節的限平行度指令
