執行順序：

- [ ] 初始化資料庫

- [ ] 註冊 1 位黑名單
- [ ] 用單張照片驗證 VIP 辨識
- [ ] 用單張照片驗證黑名單辨識

- [ ] 啟動即時辨識 GUI，做現場實測

建議命令：

```bash
python3 src/scripts/manage_vip_database.py init
python3 src/scripts/manage_vip_database.py add-vip-image "測試VIP" photo.jpg --gender M --level gold
python3 src/scripts/manage_vip_database.py add-blacklist-image "測試黑名單" photo.jpg --gender M --risk high --reason "test"
python3 src/scripts/test_vip_recognition.py path/to/vip.jpg --expect-type vip --expect-name "測試VIP"
python3 src/scripts/test_vip_recognition.py path/to/blacklist.jpg --expect-type blacklist --expect-name "測試黑名單"
python3 src/scripts/test_vip_recognition.py path/to/visitor.jpg --expect-type visitor

python3 src/scripts/realtime_detection_insightface.py
```

驗證證據至少要留：

- [ ] 單張照片測試輸出結果
- [ ] 即時辨識畫面截圖
- [ ] 誤判或漏判案例紀錄
- [ ] 當下使用的閾值（VIP / blacklist）
