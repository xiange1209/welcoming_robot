## 各套件重點

### smartnav_vision（executables: `face_recognition` / `face_registration` / `recognition_text_bridge`）

- **face_recognition_node**：訂閱 `image_topic`（預設 `/image_raw`, BEST_EFFORT QoS）；發布 `/face_recognition/result`（RecognitionResult）與可選 `/face_recognition/debug_image`；服務 `/face_recognition/refresh_cache`（std_srvs/Empty）
  - 參數：`image_topic`、`model_name`（預設 `buffalo_sc`）、`confidence_threshold`（0.5）、`enable_gpu`（true）、`recognition_threshold`（0.6）、`publish_debug_image`（false）、`result_republish_interval`（15.0 秒，同一身分去抖動）
- **face_registration_node**：服務 `/face_registration/register`（smartnav_msgs/srv/RegisterFace：`person_name`/`gender`(M/F/Other)/`person_type`(VIP/BLACKLIST)/`num_samples`）；註冊成功後自動呼叫 refresh_cache
- **recognition_text_bridge_node**：訂閱 `/face_recognition/result` → 轉自然語言 → 發布 `/user_text`；參數 `result_topic`、`user_text_topic`
- **face_engine.py**：InsightFace 封裝與標框（VIP=綠框、黑名單=紅框、訪客=藍框）；**database_manager.py** 管理人員資料與 embedding
- **RecognitionResult.msg** 欄位：`person_uuid`/`person_name`/`gender`/`person_type`(VIP/BLACKLIST/VISITOR)/`confidence`/`timestamp`/`bbox`

### smartnav_audio（executables: `voice_trigger` / `speech_recognizer` / `speech_synthesizer` / `voice_playback`）

- **voice_trigger_node**：KWS 喚醒；發布 `/audio_in`（AudioData）與 `/voice_triggered`（Bool）
- **speech_recognizer_node**：ASR；訂閱 `/audio_in` → 發布 `/user_text`（完整句）與 `/partial_text`（過程）
- **speech_synthesizer_node**：TTS；訂閱 `/speech_text` → 發布 `/audio_out`（AudioData）
- **voice_playback_node**：播放 `/audio_out`
- 模型需先執行 `scripts/download_audio_models.sh` 下載到 `src/smartnav_audio/models/`

### smartnav_llm（executable: `llm_service`；launch: `llm.launch.py`）

- **llm_service_node**：訂閱 `/user_text`（std_msgs/String）；發布 `llm_response`、`llm_stream`、`speech_text`
- 參數：`ollama_base_url`（預設 `http://localhost:11434`）、`model_name`（預設 `gemma4:e2b`）、`temperature`（0.0）
- 工具集（**全為導航導向**）：`create_map` / `list_maps` / `switch_map` / `create_waypoint` / `list_waypoints` / `navigate` / `global_localization`
- System prompt：`src/smartnav_llm/config/system_prompt.txt`（導航機器人設定）

### smartnav_navigation（executables: `map_service` / `waypoint_service` / `navigation_action`）

- `launch/brain.launch.py`：啟動三節點 + AMCL + slam_toolbox + frontier_explorer + map_server 等
- `launch/nav2.launch.py`：完整 Nav2 棧 + RViz（需 TurtleBot3 或模擬環境）
- `config/`：burger.yaml、empty_map

### smartnav_bringup（launch: `smartnav.launch.py`）

一次啟動 vision + audio + llm + navigation。Launch 參數：
`image_topic`（`/image_raw`）、`enable_gpu`（true）、`ollama_base_url`（`http://localhost:11434`）、`model_name`（`gemma4:e2b`）、`use_sim_time`（false）、`enable_vision` / `enable_audio` / `enable_llm` / `enable_navigation`（皆 true）、`enable_nav2`（**false**，需機器人本體或模擬環境才開）

## LLM 延遲實測數據（2026-07-02，已驗證）

環境：Windows 筆電 RTX 3050 Laptop 4GB VRAM、Ollama 0.24.0、每模型 1 次暖機 + 5 次正式、串流 `/api/chat`、銀行場景 system prompt、temperature 0.1、num_ctx 512。

| 模型 (Ollama tag) | 基底 | 首次輸出平均(ms) | 總生成平均(ms) | 最快(ms) | 最慢(ms) | 成功率 |
|---|---|---|---|---|---|---|
| gemma3-bank | gemma3:4b | 2303 | 4700 | 4583 | 4755 | 100% |
| gemma4e2b-bank | gemma4:e2b | 42235 | 47909 | 44838 | 50206 | 100% |
| gemma4e4b-bank | gemma4:e4b | 95066 | 107123 | 100231 | 123839 | 100% |
| qwen2.5:3b | qwen2.5:3b | 2218 | 3076 | 3017 | 3196 | 100% |

**關鍵解讀**：gemma4 模型檔 7.2GB/9.6GB 超出 4GB VRAM，Ollama 卸載到 CPU 導致慢 20~40 倍；gemma3:4b（4GB）與 qwen2.5:3b（1.9GB）能塞進 VRAM。此硬體上 qwen2.5:3b 最快、gemma3:4b 次之，gemma4 系列不適合即時互動。原始明細在 `benchmark_llm_results.json`。

## 指令大全（CLAUDE.md 只留最常用，其餘在此）

### 個別節點（RPi4）

```bash
ros2 run smartnav_vision face_recognition --ros-args -p publish_debug_image:=true
ros2 run smartnav_vision face_registration
ros2 run smartnav_vision recognition_text_bridge
ros2 launch smartnav_llm llm.launch.py ollama_base_url:=http://192.168.1.xxx:11434
ros2 launch smartnav_navigation brain.launch.py
```

### LLM 基準測試（Windows 筆電；務必用 python，python3 是空殼）

```powershell
python scripts/benchmark_llm_models.py --models gemma3-bank qwen2.5:3b --runs 5
python scripts/benchmark_llm_models.py --host http://localhost:11434 --output benchmark_llm_results.json
```

### 語音模型下載（RPi4，於專案根目錄）

```bash
bash scripts/download_audio_models.sh   # 下載 KWS/ASR/VAD/TTS 到 src/smartnav_audio/models/
```

### 檔案傳輸（Windows ↔ RPi4）

```bash
# 方式一：git（建議）— Windows: commit + push；RPi4: git pull
# 方式二：scp
scp -r "C:/Users/陳佳憲/Desktop/專題/src" xiange@192.168.1.125:~/smartnav-bot/

cd ~/smartnav-bot && colcon build --symlink-install && source install/setup.bash
```
